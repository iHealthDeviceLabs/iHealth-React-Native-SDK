(static) getBattery(mac)

Get the PO3 battery status.
  • This is an asynchronous call, it will return immediately.
  • If get successfully, following event will be emit:
    PO3Module.Event_Notify("event_notify_po3")
    The key and value will be as below:
    KeyValue
    actionPOProfileModule.ACTION_BATTERY_PO("battery_po")
    POProfileModule.BATTERY_POBattery Status, e.g. 90
  • If error happens, following event will be emit:
    PO3Module.Event_Notify("event_notify_po3")
    The key and value will be as below:
    KeyValue
    actionPOProfileModule.ACTION_ERROR_PO("error_po")
  • Attention, if you want to be notified, it is mandatory to call following method before you call this method:
    DeviceEventEmitter.addListener(PO3Module.Event_Notify, function (e: Event){});

    KeyValue

    POProfileModule.OFFLINEDATA_PO("offlineData")Offline Data Array.
    POProfileModule.DATAID("dataID")Data Id for each element in the array, e.g. 67FFC7A454148723718144A604D4152B
    POProfileModule.MEASURE_DATE_PO("measureDate")The measure data.
    POProfileModule.BLOOD_OXYGEN_PO("bloodoxygen")Blood Oxygen for each element in the array, e.g. 98
    POProfileModule.PULSE_RATE_PO("heartrate")Heart Rate for each element in the array, e.g. 65
    POProfileModule.PULSE_WAVE_PO("pulseWave")Pulse Wave for each element in the array, e.g. [0,0,0]

(static) startMeasure(mac)

Start real-time measurement.
  • This is an asynchronous call, it will return immediately.
  • If the measurement start, following event will be emit:
    PO3Module.Event_Notify("event_notify_po3")
    The key and value will be as below: <
    KeyValue
    actionPOProfileModule.ACTION_LIVEDA_PO("liveData_po")
    POProfileModule.ACTION_LIVEDA_PO("pulseWave")Pulse Wave, e.g. [2415,2371,2279]
    POProfileModule.ACTION_LIVEDA_PO("pi")PI, e.g. 0.03999999910593033
    POProfileModule.ACTION_LIVEDA_PO("pulsestrength")Pulse Strength, e.g. 6
    POProfileModule.ACTION_LIVEDA_PO("bloodoxygen")Blood Oxygen, e.g. 70
    POProfileModule.ACTION_LIVEDA_PO("heartrate")Heart Rate, e.g. 30
  • If the measurement end, following event will be emit:
    PO3Module.Event_Notify("event_notify_po3")
    The key and value will be as below: <
    KeyValue
    actionPOProfileModule.ACTION_RESULTDATA_PO("resultData_po")(iOS doesn't support this key)
    POProfileModule.ACTION_LIVEDA_PO("pulseWave")Pulse Wave, e.g. [0,0,0]

  • If error happens, following event will be emit:

    KeyValue
    actionPO3Module.ACTION_ERROR_PO("error_po")
